# D3-Media-Queries
# 🚗 Electric Car Store

Proyecto desarrollado como parte del desafío **Media Queries** de Desafío Latam.

## 📖 Descripción

Electric Car Store es una página web responsive para la visualización de vehículos eléctricos. El proyecto fue construido utilizando CSS Grid, Flexbox y Media Queries para adaptar el diseño a distintos tamaños de pantalla.

La aplicación cuenta con:

* Menú de navegación responsive
* Sección de filtros
* Catálogo de vehículos
* Footer con redes sociales
* Diseño Mobile First

## 📱 Responsive Design

El proyecto incluye dos vistas principales:

### Mobile (< 576px)

* Menú hamburguesa
* Productos en una sola columna
* Diseño optimizado para dispositivos móviles

### Desktop (≥ 576px)

* Menú lateral de navegación
* Distribución mediante CSS Grid
* Catálogo de productos en múltiples columnas
